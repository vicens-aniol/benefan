# GUIDE TO FOLLOW TO RUN THE CODE

## 1. Clone the repository

```bash
git clone https://github.com/vicens-aniol/benefan
```

## 2. Install the dependencies

```bash
cd benefan
npm install
```

## 3. Run the code

```bash
npm start
```

## 4. Scan the QR code with your Expo app

```bash
Scan the QR code with your Expo app
```
